# Guide to Game

## Interaction

Basic rules to intercation with games

- All games should be controlled by keyboard/gesture/mouse/joystick - named unified controls
- If game has action try to reproduce the original game sounds it its a clone, else create a old 8-bits for that
- All games should have a splash screen end end game screen (either loosing o wining)
- User should navigate from splash to game (starting game), and back form end game screen to splash using unified controls
- User should back to root dashboard using unified controls and a button on splash screen, even if this botton does not exists on orginal game
- Buttons on splash screen or end screen should have a small size increase when focus

## Style

- Try keep game chaos style and not bring new one


## Testing e2e

- Use playwright cli either to take a screen shots, access console logs or even play to do an e2e test
